This file contains all of the scripts and information relating to the dishwasher and plates feature of the game.

For the minimum viable product the function shows on a very simple level how a player will know if a customer has dirty plates that need washing.

As the development process continues we will see a place for dishes to be washed and a more sleek design to indicate that a customer left dirty dishes. The player will also be able to restock there clean dishes after they are washed.
